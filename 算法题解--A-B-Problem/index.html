1.题目描述

P1303 A*B Problem

题目描述

求两数的积。

输入格式

两行，两个整数。

输出格式

一行一个整数表示乘积。

输入输出样例

输入

输出

说明/提示

每个数字不超过 10^{2000}102000 ，需用高精。

2.代码实现

代码思路：

  • 整体思路是使用代码模拟人摆算式算两个数相乘。在算法开始之前我们需要排除其中一个数为0的情况。

  • 计算 a * b，就是用 a 的每一位 乘 b 的每一位。我们将计算的结果保存到一个数组中，每次累加的结果先不考虑进位。

  • a 的每一位 乘 b 的每一位，假设 a 的每一位 乘 b 的最后一位时，下一次应该是 a 的每一位 乘 b 的倒数第二位。这个模拟摆式子，需要将这一次结果保存在数组中的位置处整体前移。也就是每次 a 的每一位 乘 b的某一位后，需要控制，具体代码是：index = size_b-i;。

  • 还有就是有两处需要注意计算结果的位数，第一次是没有考虑进位，只是计算乘积的结果。第二次是考虑进位。
