Android随笔-View坐标系

Created2020-03-25|Updated2020-06-04|Android琐碎知识点
|Post View:

一. View 获取坐标的方法

  1. View.getTop()、View.getLeft()、View.getBottom()、View.getRight();

  2. View.getX()、View.getY();

  3. View.getTranslationX()、View.getTranslationY(); 等

  4. View.getLocationInWindow(int[] position);

  5. View.getLocationOnScreen(int[] position);

二. 获取View真实位置的坐标

  • 这四个获取View坐标位置的方法是相对于父ViewGroup来说的。
  • 这四个获取的值不会因为补间动画而发生改变。

三.获取View显示位置的坐标

  • 这两个获取View坐标位置的方法是相对于父ViewGroup来说的。
  • 这两个获取的值会因为补间动画而发生改变。

四.获取View属性动画的偏移值

  • 一般情况下 getX() = getLeft()，但是如果发生了属性动画，比如TranslationX。那么，getX() - getLeft() = getTranslationX()。

  • translationX是真实view 相对于看到view的 x 方向偏移量。

五.获取View相对于父窗口的坐标

  • 获取的是相对于父窗口的坐标，而不是父容器的坐标。
  • 一般情况下获取的值和View.getLocationOnScreen(position);获取的值相同。有弹窗的情况下，两者会不同。

六.获取View相对于屏幕的坐标

  • 获取的是相对于屏幕的坐标。
  • 可以利用position[1]-View.getTop())，作为一种获取状态栏高度的方法。
