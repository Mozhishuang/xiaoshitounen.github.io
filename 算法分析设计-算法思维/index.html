一. 算法设计模式

  1. 枚举 或者说 暴力求解 （Brute force）
  2. 分治法（Divide and conquer）
  3. 贪婪算法（Greedy approach）
  4. 动态规划（Dynamic programming）
  5. 回溯法（Backtracking）
  6. 分支定界法（Branch and bound）
  7. 概率算法（Randomized algorithm）

二. 算法思维

什么是算法思维？

现实问题——》形式化描述——》设计算法——》方案实施——》现实问题

算法思维解决实际问题的步骤

  1. Understand the problem（理解）
  2. Formulate it（制定）
  3. Design algorithm（设计）
  4. Implment it（实现）
  5. Run the code and solve the original problem（运行）
