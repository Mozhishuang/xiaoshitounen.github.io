一. Bitmap decode

常见的jpg，png，webp是图像的存储格式

Android要显示图片必须经过解码（decode）读取图像的数据到内存中

BitmapFactory提供了常用的一些decode方法

图片真正占用的内存要看decode之后的数据大小

二. Bitmap memory

1. <= 2.3 （API 10）

  • Bitmap解码之后的数据存储在Native Memory中
  • 手动调用Recycle回收

2. >= 3.0（API 11）

  • Bitmap解码之后的数据存储在Dalvik heap中
  • Dalvik自动回收
